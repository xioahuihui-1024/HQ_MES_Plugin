
#### 🚀 MES 助手 Pro - 核心功能一览

**1. ⚡️ 永不掉线 (Smart Keep-Alive)**

* **无感重连**：当 Session 过期时，插件会自动在后台静默登录。
* **请求重放**：即使你在过期状态下点击了“查询”，插件也会拦截失败的请求，自动登录后**重新发送**，直接展示结果。

**2. 📊 超级表格 (Advanced Table)**

* **固定表头**：向下滚动时表头自动锁定，再也不用反复回看列名。
* **自定义视图**：
    * **拖拽排序**：按住列头即可随意调整顺序。
    * **列宽调整**：像 Excel 一样拖动调整列宽。
    * **列筛选/排序**：支持当前页数据的快速筛选和升降序排列。
    * **视图记忆**：你的调整（顺序、宽度、显隐）会被自动保存，下次打开还是熟悉的样子。

**3. 👁️ 智能展示**

* **防撑开布局**：强制表格单行显示，保持界面整洁。
* **智能悬停**：鼠标划过长文本时，自动弹出黑色浮层显示完整内容，支持智能定位（不被屏幕遮挡）。
* **一键复制**：点击长文本单元格，即可弹出详情窗口并一键复制内容。

**4. 🎨 界面优化**

* **菜单高亮**：左侧菜单点击后保持高亮，页面刷新也会并自动展开父级菜单，展开选择最后一次选择菜单。
* **日期美化**：自动将 `20251210` 格式化为易读的 `2025-12-10` 等。
